# emoji-talk
emoji-talk é um chat em tempo real apenas com emojis
 
Esse codigo foi feito acompanhando a seguinte playlist (https://www.youtube.com/watch?v=ow8mA47uygc&list=PLhW5jRUibMHXY8ZaHDm_bf_54ncnee4rv&index=1&ab_channel=CodDevTV) da página CodDev TV do youtube.
 
 Nele é utilizado Mongo, Express, Vue, Node, bcrypt e JWT
